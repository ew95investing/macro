🐙 GitHub 部署 & 数据扩展

本工具已为 GitHub Pages 部署优化。上线后可通过以下方式扩展数据能力：

🤖 AI 内容生成
配置 Anthropic API Key，启用真正的 AI 生成（Claude Sonnet）。在 GitHub Secrets 设置 ANTHROPIC_API_KEY，或通过 Cloudflare Worker 转发。
🔴 微博实时数据
申请微博开放平台 App Key，配合后端接口拉取财经热搜、达人动态。或使用 RSS.app 订阅微博 RSS。
𝕏 X API 接入
配置 X API v2 Bearer Token，通过 Cloudflare Worker 做 CORS 代理，实时拉取达人推文流。
⛓ 链上数据
接入 Glassnode / Dune Analytics API，获取 BTC 链上资金流、交易所净流量、矿工持仓等专业指标。
